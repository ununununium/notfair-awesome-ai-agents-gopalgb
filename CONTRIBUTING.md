# Contributing to Awesome AI Agents

Thank you for your interest in contributing! This list is community-driven and we welcome additions.

## Guidelines

### Adding a New Entry

1. **Fork** this repository
2. **Add your resource** in the appropriate category in `README.md`
3. **Follow the format** exactly:
   - For GitHub projects: `| [Name](link) | ![Stars](https://img.shields.io/github/stars/owner/repo?style=flat-square&label=) | One-line description |`
   - For non-GitHub: `| [Name](link) | -- | One-line description |`
4. **Keep descriptions concise** -- one line, no period at end
5. **Submit a pull request** with a clear title

### Criteria for Inclusion

- **Actively maintained** -- at least one commit within the last 6 months
- **Relevant** -- must be directly related to AI agents (not general ML/AI)
- **Documented** -- must have a clear README or documentation
- **No duplicates** -- check existing entries first
- **Quality** -- should be useful to practitioners, not just proof-of-concepts

### What We Accept

- Open-source agent frameworks and libraries
- Tools for building, testing, monitoring, or deploying agents
- Notable commercial platforms (with clear value to the community)
- Academic papers that introduced foundational agent concepts
- High-quality tutorials and courses

### What We Don't Accept

- Projects with no documentation
- Abandoned projects (no activity in 6+ months)
- Self-promotional entries without community value
- Duplicate tools that don't offer anything new
- General LLM wrappers that aren't agent-specific

## Process

1. Open a pull request
2. Maintainers will review within 48 hours
3. We may suggest changes or ask questions
4. Once approved, your contribution will be merged

## Reporting Issues

If you find a broken link, outdated entry, or miscategorized item, please open an issue.

Thank you for helping keep this list awesome!
